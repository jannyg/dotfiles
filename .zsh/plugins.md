# Enhanced oh-my-zsh plugins for terminal workflow
# Add these to your plugins array in .zshrc:
# plugins=(git tmux z fzf colorize command-not-found copypath copyfile)

# This file documents the recommended plugins for Ghostty + tmux + zsh workflow:
# 
# git              - Git shortcuts and info
# tmux             - Tmux integration and shortcuts  
# z                - Jump to frequent directories
# fzf              - Fuzzy finder integration
# colorize         - Syntax highlighting for files
# command-not-found - Suggests packages for missing commands
# copypath         - Copy current path to clipboard
# copyfile         - Copy file contents to clipboard

# To use these, update your .zshrc plugins line to:
# plugins=(git tmux z fzf colorize command-not-found copypath copyfile)
